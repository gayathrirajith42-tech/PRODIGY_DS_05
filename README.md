# Traffic Accident Data Analysis

## Project Overview

This project analyzes traffic accident data to identify patterns related to accident severity, weather conditions, road surface conditions, and time of occurrence. The analysis was performed using **Python in Google Colab**, where data cleaning, exploratory data analysis (EDA), and visualizations were used to uncover key accident trends.

## Objective

* Analyze traffic accident records.
* Study accident severity patterns.
* Examine the impact of weather and road conditions.
* Identify time-based accident trends.
* Generate insights to improve road safety awareness.

## Dataset

* **Road Accident Data.xlsx**

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

* Data exploration and cleaning.
* Accident severity analysis.
* Weather condition analysis.
* Road condition analysis.
* Time-based trend analysis.
* Data visualization using charts and graphs.

## Key Findings

* Most accidents were classified as slight severity.
* Weather and road conditions influenced accident occurrence.
* Certain time periods recorded higher accident frequencies.
* Visualizations helped identify important road safety patterns.

## Conclusion

The project successfully analyzed traffic accident data and generated insights into factors affecting accidents. The complete analysis and visualizations were implemented in **Google Colab** using Python.
