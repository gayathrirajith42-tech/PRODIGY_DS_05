# PRODIGY_DS_05

## Traffic Accident Data Analysis

## Project Overview

This project focuses on analyzing traffic accident data to identify patterns related to road conditions, weather conditions, accident severity, and time of occurrence. Various data visualization techniques were used to uncover accident trends and contributing factors that may affect road safety.

## Objective

The main objective of this project is to:

* Analyze traffic accident records.
* Identify factors contributing to road accidents.
* Study accident severity patterns.
* Examine the impact of weather and road conditions.
* Visualize accident trends over time.
* Generate insights to improve road safety awareness.

## Dataset

* **Road Accident Data.xlsx**

The dataset contains information related to traffic accidents, including accident severity, weather conditions, road surface conditions, location details, and other contributing factors.

## Tools and Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

## Analysis Performed

## Data Exploration

* Loaded and explored the dataset.
* Checked data structure and column information.
* Identified missing values and data quality issues.

## Accident Severity Analysis

* Analyzed the distribution of accident severity levels.
* Visualized severity percentages using charts.

## Weather Condition Analysis

* Examined how weather conditions influence accident occurrence.
* Identified common weather conditions associated with accidents.

## Road Condition Analysis

* Analyzed the impact of road surface conditions on accident frequency.
* Compared accident occurrences under different road conditions.

## Time-Based Analysis

* Studied accident trends based on time of day.
* Identified peak accident hours.

## Data Visualization

The following visualizations were created:

* Bar Charts
* Pie Charts
* Line Charts
* Trend Analysis Graphs

## Key Findings

* Most accidents were classified as slight severity accidents.
* Accident frequency varied across different weather conditions.
* Road surface conditions influenced accident occurrence.
* Certain time periods recorded a higher number of accidents.
* Data visualization helped identify important traffic safety patterns.

## Conclusion

The project successfully analyzed traffic accident data and provided valuable insights into accident severity, weather conditions, road conditions, and time-based accident trends. These findings can contribute to a better understanding of road safety risks and support informed decision-making for accident prevention.
